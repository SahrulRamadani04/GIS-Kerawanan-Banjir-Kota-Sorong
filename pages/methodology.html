Metodologi Penelitian

Proses analisis spasial tingkat kerawanan banjir menggunakan pendekatan Multi-Parameter berbasis Geographic Information System (GIS) dengan metode Weighted Overlay.

1

Pengumpulan Data

Pengumpulan data spasial dari berbagai sumber: DEM (Digital Elevation Model), peta penggunaan lahan, jaringan sungai, dan data administrasi Kota Sorong.

2

Persiapan Data

Georeferencing, digitasi, dan konversi format data. Memastikan semua layer menggunakan sistem koordinat yang konsisten (EPSG:32752 / UTM Zone 52S).

3

Analisis Multi Parameter

Menganalisis setiap parameter penentu kerawanan banjir secara terpisah: kemiringan lereng, elevasi, penggunaan lahan, dan buffer sungai.

4

Weighted Overlay

Menggabungkan seluruh parameter dengan pembobotan tertentu menggunakan metode overlay berbobot untuk menghasilkan skor kerawanan total.

5

Klasifikasi Kerawanan

Mengklasifikasikan hasil analisis menjadi 3 tingkat kerawanan: Rendah, Sedang, dan Tinggi berdasarkan rentang skor.

6

Visualisasi GIS

Menampilkan hasil analisis dalam bentuk peta tematik interaktif berbasis web menggunakan Leaflet.js dengan data GeoJSON hasil ekspor dari QGIS.

Parameter & Bobot Analisis

Setiap parameter diberi skor berdasarkan tingkat pengaruhnya terhadap kerawanan banjir.

No Parameter Sumber Data Tipe Geometri
1 Kemiringan Lereng DEM SRTM Polygon
2 Elevasi DEM SRTM Polygon
3 Penggunaan Lahan Peta RBI Polygon
4 Buffer Sungai Peta RBI Polygon (Buffer)

Klasifikasi Tingkat Kerawanan

Hasil skor total dari overlay berbobot diklasifikasikan menjadi 3 kelas:

Kelas Warna Keterangan
Rendah Wilayah dengan risiko banjir rendah
Sedang Wilayah dengan risiko banjir moderat
Tinggi Wilayah dengan risiko banjir tinggi
